Detailed Description

Class representing a particular configuration of a scene. Only the information used during rendering is encapsulated in this class.

Member Enumeration Documentation

enum Verdi::RenderContext::DrawMode

Enumeration of possible rendering modes. Static modes should be selected when the user has uploaded the data set to GPU memory. Streaming mode should be selected if the user would rather upload the data in "bricks".
