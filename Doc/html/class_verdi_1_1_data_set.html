Detailed Description

Class for storing a data set in system memory.

Remarks:
Before uploading any data to the GPU, the system must first create or load the data from a file. This class provides a mechanism for storing and performing computation on this data. The data uses an interleaved storage format consisting of channels. No semantic information for these channels is provided - for operations that require this information, the user must provide the method with the source data and channel.
