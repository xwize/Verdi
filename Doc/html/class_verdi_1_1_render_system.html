Detailed Description

Class responsible for controlling rendering.

Remarks:
The RenderSystem allows for the registration of instances of RenderStrategy (which represent the possible rendering algorithms that can be used by the system) along with RenderContext (the render state the algorithms require to operate). Methods for switching between contexts and strategies are provided.
